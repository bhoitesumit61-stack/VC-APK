# VoxLink Flutter APK

Premium voice chat app built with Flutter + LiveKit.
Same logic as [vcrepo.vercel.app](https://vcrepo.vercel.app) — works cross-platform with web users.

## Features
- 🎙️ Real-time voice with LiveKit
- 🔒 AES-256 encrypted audio  
- 🌐 Cross-platform with web users (same room = same call)
- 🎨 Premium black & white UI with animations

## Build
GitHub Actions builds the APK automatically on every push.
Go to **Actions** → **Build VoxLink APK** → Download from **Artifacts**.
